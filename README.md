# 📊 Análisis de Datos — Alura Store Latam

## 📌 Descripción del Proyecto

Este proyecto tiene como objetivo analizar el desempeño comercial de cuatro tiendas pertenecientes a **Alura Store Latam**, utilizando técnicas de análisis de datos con Python. A partir de la información disponible, se busca generar conclusiones estratégicas que permitan apoyar la toma de decisiones del Sr. João sobre cuál tienda debería vender.

El análisis incluye métricas clave como:

* Facturación total por tienda
* Categorías de productos más vendidas
* Calificaciones de clientes
* Productos más vendidos
* Costos de envío promedio

Además, se desarrollan visualizaciones que facilitan la interpretación de los resultados y se presenta una recomendación final basada en los datos.

---

## 🎯 Objetivos

* Analizar el rendimiento financiero de cada tienda.
* Identificar patrones de ventas y comportamiento del cliente.
* Evaluar la satisfacción del cliente mediante calificaciones.
* Comparar costos logísticos entre tiendas.
* Generar visualizaciones claras para la toma de decisiones.
* Proponer una recomendación estratégica basada en evidencia.

## 📊 Análisis Realizados

El proyecto se divide en las siguientes etapas:

### 1️⃣ Análisis de Facturación

Se calculan ingresos totales, cantidad de ventas, ticket promedio y participación porcentual por tienda.

### 2️⃣ Ventas por Categoría

Se identifican las categorías con mayor volumen de ventas y su distribución entre tiendas.

### 3️⃣ Calificación Promedio

Se analiza la satisfacción del cliente mediante las calificaciones registradas.

### 4️⃣ Productos Más Vendidos

Se determinan los productos con mayor y menor rotación.

### 5️⃣ Costos de Envío

Se comparan los costos logísticos promedio entre tiendas.
